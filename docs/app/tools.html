Decision Diagnostic (DX)

Ingest a portfolio company's CSVs; surface the top-3 repeatable, high-volume decisions being made badly — each with a counterfactual, a time-stability score, and row-level evidence.

12 portcos diagnostic'd — Lending Club, Yasserh CC0, CFPB HMDA. $800k single-portco; $3.2M / $184M / $1.14B fund-level corpora.

Moat · Layer 2

Cross-Portco Benchmarking (BX)

Roll up N portcos into a single fund-level rank table, archetype index, peer groups by cosine similarity, and quarter-over-quarter persistence — the LP-letter exhibit no scorecard vendor ships.

3 fund corpora shipped — 5-region Lending Club ($3.19M), 5-state CFPB HMDA ($183.83M), 7-portco mixed-vertical ($1.14B).

Moat · Layer 2

LLM Eval for PE

Score any PE document AI output on four dimensions — citation accuracy, hallucination rate, coverage, consistency. The QA layer every other tool runs through. Catches real outliers, calibrates the rubric.

13-memo corpus run — 87% mean citation, surfaced HMDA_DE and HMDA_MA at 33% as real outliers worth investigating.

CIM Red-Flag Extractor

Fetch any SEC 10-K or S-1 by ticker. Run eight heuristic flag families with section-level citations a reviewer can verify. Compresses a 200-page disclosure into a 1-page citation-grade brief.

49 flags from real Sotera Health 10-K (SHC FY2025) — 16 high · 32 medium · 1 low. SEC EDGAR fetch, no auth.

Operate · Layer 4

100-Day Plan Drift Monitor

A frozen 100-day plan diffed against SEC EDGAR 10-Q actuals. Per-initiative drift status (on-track / lagging / off-track), dollar-gap quantification, root-cause attribution. Catches the Day-60 problem before the QBR.

5 of 7 initiatives off-track for SoteraCo (SHC) · $121.6M EBITDA at risk · dynamic-pricing pilot the biggest single miss at $85M.

Procurement Benchmarking

Cross-buyer vendor price variance from public USAspending.gov federal-contract data. Each agency treated as a portco; same-vendor price spreads ranked by total recoverable savings. Apollo's playbook, productized.

$936M cross-agency savings opportunity · 198 contracts, 28 agencies, 132 vendors. State Dept paying ~6× DHS for the same vendor.

Agent Sprawl Auditor
